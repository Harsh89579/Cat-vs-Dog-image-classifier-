# 🐱 vs 🐶 Image Classifier

[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.0+-FF6F00?style=for-the-badge&logo=tensorflow&logoColor=white)](https://www.tensorflow.org/)
[![Keras](https://img.shields.io/badge/Keras-Deep_Learning-D00000?style=for-the-badge&logo=keras&logoColor=white)](https://keras.io/)

A deep learning project that classifies images into 'Cats' or 'Dogs' using Convolutional Neural Networks (CNN). This project demonstrates the implementation of image preprocessing, data augmentation, and model serialization in Keras.

## 🧠 Model Architecture
- **Input Layer**: 128x128 RGB images.
- **Convolutional Layers**: Multiple Conv2D and MaxPooling2D layers for feature extraction.
- **Dense Layers**: Fully connected layers with Dropout for regularization.
- **Output Layer**: Sigmoid activation for binary classification.

## 🛠 Tech Stack
- **Framework**: TensorFlow / Keras
- **Data Processing**: NumPy, OpenCV
- **Analysis**: Matplotlib (for training curves)

## 📂 Repository Content
- `Cat vs dog image classifier.ipynb`: Full training pipeline and EDA.
- `Cat_vs_Dog_image_classifierr.keras`: Pre-trained model for inference.

## 🚀 How to Use
1. Clone the repo.
2. Open the `.ipynb` file in Google Colab or Jupyter Notebook.
3. Load the `.keras` file to skip training and start classifying immediately!
